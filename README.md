# 嵌入式通用化屏幕菜单

适用于嵌入式单片机设计的通用化屏幕菜单,适用于像素屏
基于已完成的屏幕驱动进行上层开发,只需对接口进行修改即可适配屏幕

# 有什么作用?

最大优势在于易用:易移植,易维护

# 菜单"插件"结构 

菜单分为多个板块,目前完成参数列表菜单板块设计

菜单分为接口,算法,逻辑处理

通过引出接口,只需对接口进行修改即可适配,移植难度大大降低

算法部分对每一个功能进行封装供逻辑部分调用,封装式设计进行解耦合,通过数学计算算出光标位置及形状

逻辑部分通过状态机调控,仅需4个输入状态即可控制整个状态机
